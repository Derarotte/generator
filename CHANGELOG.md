## [1.0.0] - 2026-01-20

### 🎉 首个稳定版本

这是中国学生作业代码生成器的第一个正式发布版本。

### 新增

- ✨ **模板驱动架构** - 代码模板与引擎分离，使用 YAML 配置 + Jinja2 模板
- ✨ **学生管理系统模块** - 完整的 Spring Boot + Vue 3 项目模板
- ✨ **FastAPI 后端** - RESTful API，支持模块查询、项目生成、文件下载
- ✨ **Vue 3 前端** - 现代化 Web 界面
- ✨ **CLI 工具** - 命令行生成支持
- ✨ **完整文档** - 模块开发指南、API 文档、部署指南

### 技术栈

- **后端**: Python 3.9+, FastAPI, Jinja2, Pydantic
- **前端**: Vue 3, Vite, Element Plus
- **模板**: YAML + Jinja2

### 系统要求

- Python 3.9+
- Node.js 18+
